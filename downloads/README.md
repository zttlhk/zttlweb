# Downloads Directory

This directory should contain the COA and TDS PDF files for all products.

Naming convention:
- COA: ProductName_CAS_COA.pdf
- TDS: ProductName_CAS_TDS.pdf

Example:
- Acetyl_Hexapeptide-8_616204-22-9_COA.pdf
- Acetyl_Hexapeptide-8_616204-22-9_TDS.pdf

Files to be provided by client before deployment.
